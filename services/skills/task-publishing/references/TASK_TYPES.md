# 12 种任务类型详细说明

| 类型 | 玩家要做什么 | 基础奖励倍率 | 额外加成 | 典型触发 |
| --- | --- | --- | --- | --- |
| 问候 | 无战斗；对话一次 | ×1 | 无 | 普通成员 NPC 日常寒暄 |
| 传话 | 无战斗；去找某 NPC 对话 | ×1 | 无 | 外交 / 商业联络 / 学术交流 |
| 通关 | 简单 / 冒险难度通关 | ×2 | 无 | 玩家问"哪里需要清理？" |
| 清理 | 简单 / 冒险难度通关 | ×2 | 无 | 同上，偏扫荡类 |
| 挑战 | 修罗 / 地狱难度通关 | ×2 | 经验值占比 ≥ 50% | 玩家问"我该去哪历练"、高阶委托 |
| 切磋 | 通关当前 NPC 的切磋关卡/副本 | ×2 | 无 | 玩家问"我要和你打" 且 NPC 有切磋能力 |
| 资源收集 | 收集食材/药剂/材料/弹夹并提交 | ×1 | + 提交品价值的 1~2 倍 | 玩家说"给你弄点材料" —— 注意玩家索要资源时禁用此类型 |
| 装备缴纳 | 获取 1 件装备并提交 | ×1 | + 提交品价值的 1~2 倍 | 战斗类 NPC 索要武器/防具 —— 玩家索要装备时禁用 |
| 特殊物品获取 | 获取 1 件特殊物品（现炒菜品/高级消耗品/插件）并提交 | ×1 | + 提交品价值的 1~2 倍 | 科技 NPC 要高级材料 |
| 物品持有 | 取得物品/情报但**不提交** | ×1 | + 持有品价值的 0.5 倍 | 情报类任务、查看合成效果 |
| 通关并收集 | 去指定关卡/区域搜刮并提交 | ×2 | + 提交品价值的 1~2 倍 | "去医院收集抗生素吧" |
| 通关并持有 | 去指定关卡/区域寻找物品但不提交 | ×2 | + 持有品价值的 0.5 倍 | "去废城看看那份情报还在不在" |

## 收集任务选型优先级

- 玩家提到"去某地/某区域"：`通关并收集` > `资源收集`（必须选通关并收集）
- 玩家没提地点：`通关并收集` 与 `资源收集` 同级，可任选

## 硬禁区

- 玩家**索要资源** → 禁止选 `资源收集`（把需求放 `rewards`）
- 玩家**索要装备** → 禁止选 `装备缴纳`（把需求放 `rewards`）
- 玩家**索要药剂** → 禁止选 `资源收集`（药剂放 `reward_types.optional`）

## NPC 身份 × 推荐任务类型

| NPC 类型 | 推荐任务类型 | 奖励偏好 |
| --- | --- | --- |
| 高级 NPC | 传话(外交) / 通关 / 清理 / 挑战 / 通关并收集 / 通关并持有 | 金币 / 经验值 / 武器 / 强化石 / 技能点 |
| 商人 NPC（有商店） | 传话(商业联络) / 装备缴纳 / 特殊物品获取 / 物品持有 / 资源收集 | 金币 / 商店物品 / 材料 |
| 普通成员 NPC | 问候 / 传话 / 低级资源收集 / 低级物品持有 | 金币 / 药剂 / 食品 / 弹夹 |
| 科技/学术 NPC | 传话(学术交流) / 特殊物品获取(高级材料/插件) / 物品持有(情报) / 装备缴纳 | 金币 / K 点 / 技能点 / 插件 / 合成材料 / 强化石 / 战宠灵石 |

推荐仅供参考，结合你的具体身份灵活判断即可。
